# React Native Tetris (Minimalist Edition)

A clean, modern Tetris clone built with **React Native** and **Expo**. This project features a custom-built game engine using React Hooks, a minimalist UI, and touch-optimized controls.

## ðŸ“± Features

* **Classic Gameplay:** Full implementation of the 7 classic Tetromino shapes and rotation rules.
* **Minimalist Design:** Clean 10x20 grid with a distraction-free interface.
* **Touch Controls:** Ergonomic, large circular buttons for Left, Right, and Rotate actions.
* **Game Loop:** Custom `useInterval` hook for accurate game timing and gravity.
* **Scoring System:** Tracks score based on lines cleared.
* **Navigation:** Stack navigation between Main Menu and Game Screen.
